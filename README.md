# RFM-Based Customer Segmentation & Online Retail Dashboard

## 📋 Project Overview
End-to-end **RFM (Recency, Frequency, Monetary)** analysis performed on a **541,000-row online retail dataset** (2010–2011) using Microsoft Excel. Customers were scored on a 1–5 scale across all three dimensions and segmented into meaningful business categories: **Champion, Loyal Customers, Potential Loyalist, At Risk, Hibernating, Lost, and New**.

## ✨ Features & Analysis
- Full data cleaning and transformation using Power Query
- RFM scoring logic (Recency, Frequency, Monetary on 1–5 scale)
- Customer segmentation into 7 business segments
- Revenue trends by month, country, and product
- Interactive PivotTables and dashboard for slicing by segment, time, and geography
- Identification of high-value vs. at-risk customers

## 🛠 Tools & Techniques
- **Microsoft Excel** (Advanced PivotTables, Power Query, formulas)
- RFM segmentation methodology
- Data visualization with charts and dashboard layout
